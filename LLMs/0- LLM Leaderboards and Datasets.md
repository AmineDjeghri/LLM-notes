## Easy guides
- https://www.evidentlyai.com/llm-guide/llm-benchmarks

LLM benchmarks are **sets of tests** that help assess the capabilities of a given LLM model.
My def : the evaluation of models on a given set of tests (dataset)

LLM benchmarks vary in difficulty. Early ones focused on basic tasks like classifying text or completing sentences, which worked well for evaluating smaller models like BERT.
